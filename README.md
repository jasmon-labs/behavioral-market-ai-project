Behavioral Market AI Dashboard

Overview
- Predicts NIFTY market behavior using XGBoost
- Incorporates market sentiment analysis from financial news
- Visualizes predictions through a Streamlit dashboard

Features
- Historical NIFTY trend analysis
- Sentiment analysis using VADER
- XGBoost multiclass classification
- Feature importance visualization
- Market behavior prediction dashboard

Tech Stack
- Python
- Pandas
- Scikit-learn
- XGBoost
- Streamlit
- Plotly

Results
- Accuracy: 46.6%
- Classes: Bullish, Stable, Panic, High Volatility

How to Run
1. Install requirements
2. Run preprocessing.py
3. Run model_training.py
4. Run dashboard.py using Streamlit
